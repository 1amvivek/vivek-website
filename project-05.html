Tap2Pay Money Transactions using Near-field Communications

close

Description:

Tap2Pay is a final year project in my bachelors which explores new ways in payment transaction system through NFC enabled mobile phones. It uses Near-field Communication (NFC) technology to exchange payment information between the users and then updates the same to the respective bank servers about the transaction. The objective of the project is to reduce the usage of actual currency and start digitizing the payments in the lowest of levels possible without using an external device other than the ones we already have.
